# 📥 Universal Video Downloader

Простой скрипт для скачивания видео и аудио с сотен платформ — YouTube, VK, Twitch, TikTok, Twitter/X и [многих других](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md).

---

## 🌟 Возможности

- Скачивание по ссылке или по названию (без URL — просто введите что ищете)
- Пакетная загрузка: несколько ссылок/запросов через запятую за раз
- Выбор качества: `best`, `1080p`, `720p`, `480p`, `worst`
- Режим аудио — сохраняет трек в `.mp3` (192 kbps)
- Смена папки загрузки прямо во время работы, без перезапуска
- Прогресс загрузки: процент, скорость и оставшееся время
- Лог-файлы с историей загрузок (по дате)
- Два режима: **интерактивный** (диалог в терминале) и **пакетный** (аргументы в командной строке)

## ✨ Дополнительно (по желанию)

### FFmpeg
Без FFmpeg скрипт работает, но скачивает видео в формате `mp4` с ограниченным выбором качества.  
С FFmpeg — видео и аудио скачиваются раздельно в максимальном качестве и склеиваются автоматически. Также требуется для сохранения аудио в `.mp3`.

Скачать: https://ffmpeg.org/download.html

После установки убедитесь, что `ffmpeg` доступен в `PATH` — скрипт определит его наличие сам и сообщит при запуске.

## 🚀 Запуск

### Интерактивный режим
```bash
python video_downloader.py
```

### Пакетный режим
```bash
# Одна ссылка
python video_downloader.py "https://youtu.be/..."

# Несколько ссылок с указанием качества и папки
python video_downloader.py "https://youtu.be/..." "https://youtu.be/..." -q 720 -o "C:/Videos"

# Поиск по названию + только аудио
python video_downloader.py "lofi hip hop radio" --audio
```

### Аргументы
| Аргумент | По умолчанию | Описание |
|---|---|---|
| `urls` | — | Ссылки или поисковые запросы |
| `-o` / `--output` | `downloads` | Папка для сохранения |
| `-q` / `--quality` | `best` | Качество: `best` `1080` `720` `480` `worst` |
| `--audio` | выкл | Только аудио `.mp3` |
| `--log-dir` | `logs` | Папка для лог-файлов |

---

## ⌨️ Команды интерактивного режима

| Команда | Действие |
|---|---|
| `:quality <best\|1080\|720\|480\|worst>` | Сменить качество |
| `:audio` | Включить / выключить режим аудио |
| `:path` | Показать текущую папку загрузки |
| `:path <папка>` | Сменить папку загрузки |
| `:ffmpeg` | Проверить статус FFmpeg |
| `:stats` | Счётчик загрузок за сессию |
| `:help` | Справка по командам |
| `выход` / `exit` / `q` | Завершить работу |
